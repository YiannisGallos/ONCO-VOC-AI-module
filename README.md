# ONCO-VOC-AI-module
ONCO-VOC is a rapid, non-invasive breath biopsy platform for colorectal cancer detection that employs a 48-gold-nanoparticle sensor array to analyze exhaled VOC patterns, integrates machine learning for high-accuracy classification with explainable outputs, and has demonstrated robust clinical performance across multi-site trials while enabling early risk indication and potential screening improvements.

This repository contains the code and part of the dataset that has been used as part of the ONCOSCREEN EU horizon project.
My work is localized specifically on the use of AI algorithms on datasets from ONCO-VOC device for the timely diagnosis of Colorectal Cancer.
